# Yandex Music Track Collector+

> Browser console script to export tracks from Yandex Music web playlists (e.g. "Liked" playlist) to TXT/CSV files.

This script injects a small floating panel into the Yandex Music web UI and collects track metadata as you scroll through the playlist. It was tested on large "Liked" playlists and is robust enough to handle some non-standard track rows via a fallback parser.

---

## Track Collector+ для Яндекс Музыки

Скрипт для экспорта треков из плейлистов веб-версии Яндекс Музыки через консоль браузера.

Основной сценарий — выгрузить плейлист «Мне нравится» в формат TXT или CSV, чтобы дальше работать с ним где угодно (анализ, перенос в другой сервис, резервная копия и т.п.).
